WordPress Local Stream Wrapper Base Class

This file contains the local WordPress stream wrapper base class. This is a simple implementation of WP_Stream_Wrapper_Interface and can be used by stream wrappers that manipulate files on the local filesystem. A custom wrapepr such as "local://" can be created with just a few lines of code.

Developers: If you are using this class as a base for your own wrapper you must make sure that this class exists first. It is quite possible that an end-user might install your wrapper without having first installed the WP Stream Wrappers plugin. If this happens, everything will come to a crashing halt. Refer to the included WP Local Stream Wrapper for an example on how to handle this problem. Basically, put your implementation class in a separate file and only include it upon registration.
