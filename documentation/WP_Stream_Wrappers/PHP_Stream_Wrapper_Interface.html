Interface PHP_Stream_Wrapper_Interface

Description
Description | Descendents | Methods (details)

Generic PHP stream wrapper interface

A simple prototype of the stream wrapper class used by the WordPress stream wrapper interface. Refer to WP_Stream_Wrapper_Interface for further documentation on resources and methods.

Note: This interface should not be directly implemented. Instead, extend WP_Stream_Wrapper_Interface.
