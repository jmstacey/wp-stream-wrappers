wp_chmod (line 39)

Changes permissions of the resource

PHP's own chmod() function has trouble with streams, so we implement the correct behavior ourselves in this replacement helper function.

This function is fully compatible with PHP's chmod() function and can be called in the same way. For example, both a URI and a normal filepath can be provided for the $uri parameter.

  • return: true on success or false on failure.
  • link: http://php.net/manual/en/function.chmod.php
  • since: 1.0.0
bool wp_chmod (string $uri, [int $mode = null])
  • string $uri: the stream URI, or a filepath to the file.
  • int $mode: the octal number components specifying access restrictions. Refer to the PHP documentation for more information.

wp_rmdir_recursive (line 240)

Removes directory recursively

IMPORTANT: This function is experimental and has not been tested.

Attempts to remove the given directory recursively. Unlike PHP's rmdir() which only removes the directory if it's empty, wp_rmdir_recursive() will try to recursively delete all files and directories. Essentially this is equivalent to a "rm -rf" command, or a forced rmdir() if you will.

  • return: true on success or false on failure.
  • see: rmdir()
  • link: http://us2.php.net/manual/en/function.rmdir.php
  • since: 1.0.0
bool wp_rmdir_recursive (string $uri)
  • string $uri: the URI or path to file.

wp_touch (line 197)

Sets access and modification time of file

PHP's touch() does not work well with stream wrappers. This helper function adds this missing support.

This function is fully compatible with PHP's touch() function and may be called in the same way. For example, both a URI and a normal filepath can be provided for the $uri parameter.

  • return: true on success or false on failure.
  • see: touch()
  • link: http://php.net/manual/en/function.touch.php
  • since: 1.0.0
bool wp_touch (string $uri, [int $time = null], [int $atime = null])
  • string $uri: the URI or path to file being touched.
  • int $time: The touch time. If $time is not provided, the current system time is used.
  • int $atime: If present, the access time of the given filename is set to the value of $atime. Otherwise, it is set to $time.
