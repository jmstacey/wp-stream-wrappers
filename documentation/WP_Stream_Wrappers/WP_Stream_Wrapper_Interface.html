dirname (line 158)

Returns the directory name component of given path

PHP's dirname() function does not support stream wrappers. This default is provided so that individual wrappers may implement their own solutions. See WP_Local_Stream_Wrapper_Base for an example.

get_web_accessible_url (line 196)

Returns the web accessible URL for the resource

Returns a URL that can be accessed from a browser. For example, the web URL of the internal URI "local://example.txt" might be "http://www.example.com/wp-content/example.txt".

All wrappers that intend to make their resources available in this way must implement this method.

  • return: the web accessible URL for the resource.
