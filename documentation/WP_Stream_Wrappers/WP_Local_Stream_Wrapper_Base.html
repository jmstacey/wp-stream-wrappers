WordPress Local Stream Wrapper Base

A simple implementation of WP_Stream_Wrapper_Interface and ready to use skeleton for manipulating files on the local file system. New custom tailored stream wrappers that deal with local files can be created with just a few lines of code.

Look at the WP Local Stream Wrapper (WP_Local_Stream_Wrapper) for an example of how to use this base class.
